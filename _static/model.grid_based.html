vector field display of the loads applied to the chosen set of elements in the load case passed as parameter Parameters:

viewNm: name of the view that contains the renderer (possible
options: “XYZPos”, “XPos”, “XNeg”,”YPos”, “YNeg”, “ZPos”, “ZNeg”)
fileName: full name of the graphic file to generate. Defaults to
