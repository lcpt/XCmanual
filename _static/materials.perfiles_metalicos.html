materials.perfiles_metalicos.mallaSetConPerfilMetalico module¶

Genera elementos de tipo «ElasticBeam3d» para las líneas
del conjunto que se pasa como parámetro. El conjunto ha de tener definidos las siguientes variables: - nmbPerfil: Nombre del perfil a emplear (por ejemplo L50x50x5) - nmbGeomTransf: Nombre de la transformación geométrica a emplear. - numDivisiones: Número de elementos a generar en cada línea.
