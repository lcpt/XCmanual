class materials.sections.fiber_section.def_simple_RC_section.RecordRCSlabBeamSection(name, sectionDescr, concrType, reinfSteelType, depth, width=1.0, elemSetName='total')¶

Bases: materials.sections.fiber_section.def_simple_RC_section.setRCSections2SetElVerif

This class is used to define the variables that make up the two reinforced concrete sections that define the two reinforcement directions of a slab or the front and back ending sections of a beam element

Variables:
  • name – basic name to form the RC sections in direction 1 (name+‘1’) and direction 2(name+‘1’)
  • sectionDescr – section description
  • concrType – type of concrete (e.g. EHE_materials.HA25)
  • reinfSteelType – type of reinforcement steel
  • depth – cross-section depth
  • width – cross-section width (defaults to 1.0)
  • dir1PositvRebarRows – layers of main rebars in direction 1 in the local positive face of the section (list of MainReinfLayer)
  • dir1NegatvRebarRows – layers of main rebars in direction 1 in the local negative face of the section (list of MainReinfLayer)
  • dir2PositvRebarRows – layers of main rebars in direction 2 in the local positive face of the section (list of MainReinfLayer)
  • dir2NegatvRebarRows – layers of main rebars in direction 2 in the local negative face of the section (list of MainReinfLayer)
  • dir1ShReinfY – instance of class RecordShearReinforcement that represents the Y shear reinforcement in section 1
  • dir1ShReinfZ – instance of class RecordShearReinforcement that represents the Z shear reinforcement in section 1
  • dir2ShReinfY – instance of class RecordShearReinforcement that represents the Y shear reinforcement in section 2
  • dir2ShReinfZ – instance of class RecordShearReinforcement that represents the Z shear reinforcement in section 2
  • elemSetName – name of the set with the elements to which to assign the section (defaults to ‘total’).
