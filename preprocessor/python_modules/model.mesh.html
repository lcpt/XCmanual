Mesh utils¶

class model.mesh.finit_el_model.LinSetToMesh(linSet, matSect, elemSize, vDirLAxZ, elemType='elastic_beam_3d')¶

Bases: object

Define the parameters to mesh a set of lines. The method generateMesh meshes those lines and adds the nodes and elements created to the set .

Variables:
  • linSet – set of lines.
  • matSect – instance of the class BeamMaterialData that defines the material-section to be applied to the set of lines.
  • elemSize – mean size of the elements
  • vDirLAxZ – direction vector for the element local axis Z defined as xc.Vector([x,y,z]). This is the direction in which the Z local axis of the reinforced concrete sections will be oriented (i.e. in the case of rectangular sections this Z local axis of the RC section is parallel to the dimension defined as width of the rectangle)
  • elemType – type of element for the mesh (defaults to ‘elastic_beam_3d’)
